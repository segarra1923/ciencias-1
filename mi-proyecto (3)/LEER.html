¡Hola!

Este directorio contiene tu juego generado con pilas-engine listo para empaquetar o distribuir. Este archivo en particular solo contiene instrucciones iniciales.

Pero antes de usarlo, algunas aclaraciones importantes:

  • La carpeta "proyecto" tiene todos los archivos que necesita tu juego para funcionar: imágenes, fuentes, animaciones, bibliotecas etc...
  • Necesitas un webserver para ejecutar el juego de forma local. Mirá esta documentación para más detalles: Cómo exportar juegos
  • Para generar un ejecutable .exe de tu juego, una aplicación mobile o subirlo a web mirá esta documentación con consejos e instrucciones: Cómo exportar juegos
  • Si tienes Node.js instalado en tu equipo, simplemente ingresá en la carpeta "proyecto" y ejecuta: 
    para instalar dependencias:

  npm install

para ejecutar en un navegador local:

  npm start

para publicar en los servidores gratuitos de https://surge.sh:

  npm run surge

para probar offline:

  npm run electron
